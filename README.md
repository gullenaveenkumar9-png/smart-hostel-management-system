# Smart Hostel Management System

A full-stack web application for efficient hostel administration in educational institutions.

## Features

- Student Management
- Room Allocation
- Leave Management
- Complaint Tracking
- Attendance Monitoring
- Visitor Management
- Fee Management
- AI Chat Assistant
- Analytics Dashboard

## Tech Stack

- React
- TypeScript
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Installation

1. Clone the repository
2. Install dependencies
   npm install
3. Configure environment variables
4. Run the application
   npm run dev
